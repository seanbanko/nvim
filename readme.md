## Install Instructions

 > Install requires Neovim 0.9+. Always review the code before installing a configuration.

Clone the repository and install the plugins:

```sh
git clone git@github.com:seanbanko/dotfiles ~/.config/seanbanko/dotfiles
NVIM_APPNAME=seanbanko/dotfiles/nvim nvim --headless +"Lazy! sync" +qa
```

Open Neovim with this config:

```sh
NVIM_APPNAME=seanbanko/dotfiles/nvim nvim
```
